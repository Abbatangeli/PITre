INSTALLAZIONE:

in Microsoft IIS SERVER:
creare le seguenti Virtual Directory:

1. Frontend Apppool FRamework4.0 copiare al suo interno il build in publish del progetto \PiTre3.2.19\NttDataWA

2. Backend Apppool Frameowrk 3.5 copiare al suo interno il build in publish del progetto \PiTre3.2.19\Vt-Docs\DocsPaWS

3. Admintool Appool FRamework 2.0 e copiare al suo interno il progetto  \PiTre3.2.19\Vt-Docs\DocsPaWA

editare il web.config del progetto nttdatawa e admintool in modo che la chiave della url del

    <add key="LANGUAGES_PATH" value="<path fisico dell'installazione\NttDataWA\Languages\%DATA"/>

e 
<applicationSettings>
    <NttDataWA.Properties.Settings>
      <setting name="NttDataWA_DocsPaWR1_DocsPaWebService1" serializeAs="String">
        <value>http://<path server >/DocsPaWS/DocsPaWS.asmx</value>
      </setting>
      <setting name="NttDataWA_DocsPaWR1_DocsPaWebService" serializeAs="String">
        <value>http://<path server >/DocsPaWS/DocsPaWS.asmx</value>
      </setting>
      <setting name="NttDataWA_DocsPaWR_DocsPaWebService" serializeAs="String">
        <value>http://<path server >/pat-be/DocsPaWS.asmx</value>
      </setting>
    </NttDataWA.Properties.Settings>
  </applicationSettings>

editare il web.config del progetto DocspaWs in modo da inserire la corretta stringa di connessione verso il server db ORACLE utilizzato.

installare il servizio Centro notifiche:

editare il web.config del progetto \PiTre3.2.19\Pitre-AltriProgetti\ServiceNotifications
con la stringa di connessione al proprio server oracle.
Successivamente installare il servizio Windows

\PiTre3.2.19\Pitre-AltriProgetti\ServiceNotifications, lanciando da Shell Windows come amministratore il seguente comando:

install.bat
