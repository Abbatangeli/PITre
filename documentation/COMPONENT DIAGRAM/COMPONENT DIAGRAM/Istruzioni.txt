Conponent Diagram:
1. La cartella 'PDF' contiene l'export in formato PDF del Component Diagram. Utilizzare uno zoom adeguato nel viewer PDF per una corretta visualizzazione del Diagramma.

2. La cartella 'VP' contiene l'export del progetto in Visual Paradigm, software utilizzato per realizzare il component Diagram

3. La cartella 'XML' contiene l'export in formato XML del progetto per essere aperto con un altro programma che supporti l'importazione da XML

4. La cartella 'XMI' contiene l'export in formato XMI del progetto per essere aperto con un altro programma che supporti l'importazione da XMI (i.e. Visio)
