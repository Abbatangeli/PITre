Schema ER:

1. La cartella 'ER-Completo' ed il file 'ER-Completo.dmd' sono i file di progetto che caratterizzano lo schema ER. Per aprirlo utilizzare il software SQL Developer.

2. La cartella 'EXPORT PDF' contiene le estrazioni in formato PDF dello schema ER in maniera globale (Tutto lo schema ER) e settoriale per tipologie (Viste dello schema ER). E' consigliabile utilizzare un adeguato Zoom del viewer PDF per una corretta visualizzazione dello schema.
